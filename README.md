# Metode-Validasi-Statistika-pada-Suatu-Data-
Repository ini berisi implementasi metode validasi statistika menggunakan Python, meliputi uji normalitas (Jarque-Bera, Kolmogorov-Smirnov), homogenitas varians (F-Test, Levene Test), dan independensi (Chi-Square, Fisher's Exact Test) untuk mendukung analisis dan evaluasi kualitas data.
